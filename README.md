# My Portfolio Website

A multi-page personal portfolio built during Week 1 of my Full-Stack Web Development internship, focused on semantic, accessible HTML5.

## Pages
- **Home** — introduction and featured projects
- **About** — my background and journey
- **Projects** — a showcase of work completed so far
- **Skills** — technologies I'm learning and using
- **Contact** — a form and direct contact links

## What I practiced this week
- Semantic HTML5 elements (header, nav, main, article, section, aside, footer)
- Accessible forms with native validation
- Git version control: commits, branches, and pull requests

## How to view
Open `index.html` in a browser, or run it locally with the Live Server VS Code extension.
